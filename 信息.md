# MemeFlow 项目大览

## 项目基本信息

- **项目名称**: MemeFlow (随机表情包)
- **版本**: 1.3.2
- **功能定位**: 个人表情包启动器 - 一键复制/分享,即时反应
- **开源协议**: MIT
- **作者**: sideonkeibulllll
- 备注: 仅需安卓版本开发就行

## 技术架构

### 平台支持

| 平台          | 技术栈                         | 状态     |
| ----------- | --------------------------- | ------ |
| Windows 桌面  | Electron + Vanilla JS       | ✅ 已完成  |
| Android 移动端 | Capacitor + Android WebView | ✅ 已完成  |
| iOS 移动端     | Capacitor + iOS WebView     | 📋 规划中 |

### 核心技术

- **桌面应用**: Electron 40.8
- **跨平台框架**: Capacitor 8.2
- **前端实现**: Vanilla JavaScript (零框架,无依赖)
- **数据存储**: JSON 文件 (本地文件系统)
- **构建打包**: electron-builder, Python 构建脚本

## 核心功能模块

### 1. 表情包展示

- **网格布局**: 支持 1×1 到 6×4 的自定义网格
- **随机算法**: 加权随机算法,高频使用表情优先显示
- **纯随机模式**: 多格显示时首个格子纯随机

### 2. 标签与分类

- **标签系统**: 为每张表情包添加文本标签
- **情感分类**: 四大类别
  - 快乐/滑稽 (happy)
  - 悲伤/委屈 (sad)
  - 震惊/愤怒 (shock)
  - 阴阳怪气/社交 (social)

### 3. 搜索与筛选

- **标签搜索**: 根据标签文本搜索
- **分类筛选**: 按情感类别过滤
- **排序功能**: 最多使用/最近使用

### 4. 数据管理

- **权重追踪**: 记录每张表情包的使用频率
- **使用记录**: 统计使用次数和最后使用时间
- **配置导入导出**: JSON 格式配置文件

### 5. 文件导入

- **ZIP 导入**: 批量导入表情包
- **格式支持**: JPG, JPEG, PNG, GIF, WebP

### 6. 历史记录 (移动端)

- **页面历史**: 保存最近 10 页浏览记录
- **前进后退**: 长按刷新按钮返回历史页面

## 数据存储结构

### 文件位置

- **桌面端**: `%APPDATA%/MemeFlow/` (Electron userData)
- **移动端**: 应用私有目录

### 数据文件

| 文件名               | 内容   | 说明      |
| ----------------- | ---- | ------- |
| meme-weights.json | 权重数据 | 表情包使用权重 |
| meme-tags.json    | 标签数据 | 标签文本和分类 |
| meme-config.json  | 应用配置 | 网格布局等设置 |
| meme-index.json   | 索引数据 | 表情包文件列表 |
| meme-usage.json   | 使用统计 | 使用次数和时间 |
| assets/           | 资源目录 | 表情包图片文件 |

## 构建与部署

## 架构设计理念

### 1. 零框架前端

- 使用 Vanilla JS,避免框架开销
- 最小化包体积,快速加载
- 直接 DOM 操作,无虚拟 DOM

### 2. JSON 本地存储

- 无需数据库依赖
- 数据可移植,易于备份
- 简单的版本控制

### 3. IPC 通信架构

- Electron 主进程处理文件操作
- 渲染进程专注 UI 交互
- 清晰的进程间通信

### 4. 加权随机算法

- 基于使用频率的权重分配
- 前缀和数组优化查询性能
- 二分查找快速定位

## 技术债务与改进方向

### 当前限制

1. 缺乏单元测试覆盖
2. 移动端功能与桌面端不完全同步
3. 配置导入导出需要手动选择文件
4. 缺乏云同步机制

### 改进建议

1. 引入 Jest/Mocha 测试框架
2. 统一桌面端和移动端代码库
3. 集成云存储 SDK (如 Cloudflare R2)
4. 添加自动备份和恢复功能

## 总结

MemeFlow 是一个设计简洁、功能实用的表情包管理工具。采用零框架的轻量级架构,通过 JSON 本地存储实现数据管理,支持 Windows 和 Android 双平台。项目的核心价值在于加权随机算法带来的个性化体验,以及简洁易用的标签分类系统。

未来发展方向应聚焦于:

1. 完善跨平台一致性
2. 增强数据管理能力 (云同步)
3. 提升可扩展性 (插件系统)
4. 扩大平台覆盖 (iOS 支持)
